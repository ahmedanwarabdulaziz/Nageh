# منصة إدارة الحملة الانتخابية للنادي

منصة ويب مبنية بـ Next.js تهدف إلى تمكين فريق الحملة الانتخابية من إدارة الأعضاء، تنظيم الفرق، ومتابعة حالة التصويت لحظياً، مع واجهة عربية كاملة وألوان أساسية بالأسود والذهبي والأبيض.

## البدء محلياً

1. انسخ ملف البيئة:
   ```bash
   copy config\\env.example .env.local
   ```
2. حدّث قيم المفاتيح ضمن `.env.local` (خاصة بيانات مشرف Firebase).
3. ثبّت الاعتمادات:
   ```bash
   npm install
   ```
4. شغّل خادم التطوير:
   ```bash
   npm run dev
   ```

## البنية الأساسية

- `src/app`: صفحات Next.js (App Router) تدعم اتجاه الكتابة من اليمين إلى اليسار.
- `src/components`: مكونات الواجهة، بما في ذلك تخطيط لوحة التحكم ومكونات التحليلات.
- `src/lib/firebase`: تهيئة Firebase للعميل والخادم.
- `src/styles/globals.css`: أنماط Tailwind العامة مع تعريف الألوان والخطوط.
- `config/env.example`: قالب لملف البيئة.

## الخطوات القادمة

- ربط صفحات لوحة التحكم ببيانات Firestore الحقيقية.
- إضافة حماية الصلاحيات باستخدام مطالبات مخصصة من Firebase.
- تطوير وحدة استيراد ملف Excel (`ASC .xlsx`) لملء قاعدة الأعضاء.
- إنشاء مخططات التحليلات ولوحات المتابعة الفعلية.
